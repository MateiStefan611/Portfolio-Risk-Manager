# 📊 Portfolio Risk Manager

A financial dashboard built with Streamlit for portfolio risk analysis and stock evaluation.

## Features

- Portfolio volatility & annualized return
- Sharpe ratio calculation
- Correlation heatmap
- RSI indicator
- Fundamental analysis (P/E, PEG, margins, etc.)

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- yfinance

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
